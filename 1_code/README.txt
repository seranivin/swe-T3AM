# swe-T3AM
1. When going to the website, enter by clicking the 'Login' button on the dashboard.
2. You can either log in with username: test and password: test and clicking the login button to see a test file with previous tasks in it or create an account by putting your own username and password and clicking create account.
3. Add tasks using the 'Add' function, time by clicking the clock, and date by clicking anywhere on the box with the dashes. Tasks can be added to completed either when the due date passes or the checkbox is clicked.


Run the project from swe-web.html to see websites home page. All files need to be downloaded to see full website.
